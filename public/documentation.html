myFlix API Documentation Page

This project is about creating a API server which can be used by users to receive information on movies, directors, and genres for selected movies.
Users can also create a profile and save data about their favorite movies.

Business Logic URL HTTP Method Query parameters Request body data format Response body data format
Register new user /users POST None data about the user to add in json. Example: 
              
{ 
  "Username": "TestUser",
  "Password": "SECRET",
  "Email": "testing@mail.com"
}
details about added user including a generated id. Example: 
              
{ 
  "_id": "123232-33435-232",
  "Username": "TestUser",
  "Password": "SECRET",
  "Email": "testing@mail.com",
  "FavoriteMovies": []
}
Login with username and password /login?Username=[:username]&Password=[:password] POST :username, :password None a message about successful login and a Bearer Authorization Token. this token is needed to use the other end points
List all movies /movies GET None None a JSON array of all movies
Show details of a movie by title /movies/[:title] GET :title None details of the selected movie in json. Example: 
              
{
  "Title": "", 
  "Genre": {
    "Name": ""
  },
  "Director": {
    "Name": ""
  }
}
Show details for a genre /movies/genres/[:genre] GET :genre None details about the selected genre in json. Example: 
              
{ 
  "Name": "Drama"
}
Show a director's details by name /movies/directors/[:name] GET :name None details about the selected director. Example: 
              
{ 
  "Name": "Larry Davidson"
}
Update user by name /users/name/[:name] PUT :name updated user details as json. Example: 
              
{ 
  "Name": "Kingkong"
}
updated user details in json. Example: 
              
{ 
  "_id": "1",
  "Username": "Kingkong", 
  "FavoriteMovies": [
    "uyuyuiui56678ff5"
  ] 
}
List all users /users GET None None list of all users for testing
Show user by name /users/[:name] GET :name None show details of a single user by name
Show user by id /users/[:id] GET :id None show details of a single user by id
Add a movie to a user's list of favorites /users/[:name]/[:movieID] PATCH :name, :movieID None user details with the movie added
Remove a movie from a user's list of favorites /users/[:name]/[:movieID] DELETE :name, :movieID None user details with the movie removed
Remove a user by name /users/name/[:name] DELETE :name None a message confirming the user was deleted
